1. **Ranked List of Apps**

     - Rank

2. **Ranking Criteria**
   - Each entry should include:
     - Name (criterion name)

**JSON Output Format:**
```json
{
  "apps_ranked": [
    {
      "rank": 1,
      "name": "App Name",
      "features": ["Feature 1", "Feature 2", "Feature 3", ...]
    }
  ],
  "ranking_criteria": [
    {
      "name": "Criteria Name",
      "type": "Criteria Type",
      "metrics": ["Metric 1", "Metric 2", ...],
      "data_sources": ["Source 1", "Source 2", ...]
    }
  ]
}
```

Ensure that the ranking is justified using concrete data sources, and provide the most up-to-date information available.
